# Meu Portfólio

Bem-vindo ao **Meu Portfólio**, um projeto desenvolvido com [Next.js](https://nextjs.org) para explorar e visualizar informações sobre minha trajetória profissional.

🔗 **Acesse o projeto online:** [lucasluann.vercel.app](https://lucasluann.vercel.app/)

## 🚀 Tecnologias Utilizadas

- **[Next.js](https://nextjs.org)** - Framework React para aplicações web modernas.
- **[Tailwind CSS](https://tailwindcss.com/)** - Estilização rápida e eficiente.
- **[Vercel](https://vercel.com/)** - Hospedagem e deploy automático.
